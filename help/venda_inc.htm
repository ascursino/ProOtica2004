Cadastrando uma venda
  A tela abaixo é o início do cadastro de uma venda. Uma venda pode estar ou não relacionada a um cliente. Para relacionar um cliente a sua venda, basta clicar 2 vezes no nome do cliente que aparece na listagem, caso esse já esteja cadastrado no sistema, ou clicar no botão INCLUIR CLIENTE para fazer um novo cadastro de cliente e subsequentemente o cadastro de receitas (opcional) e finalmente, o cadastro da venda. Caso não seja necessário cadastrar um cliente para uma determinada venda, basta clicar no botão VENDA RÁPIDA, neste caso a venda só poderá ver à vista e não tem a necessidade de se cadastrar um cliente nem uma receita para fazer a mesma.
   
 
   
 

A tela abaixo é um exemplo de cadastro de vendas. Basta preencher os campos mostrados nesta tela e para finalizar a inclusão desta venda, clique no botão OK.

O botão VER PRODUTO, ao ser clicado, mostrará uma listagem com todos os produtos. Para escolher um produto para está venda basta clicar 2 vezes sobre o nome do produto que está na listagem exibida.

O botão LIMPAR PRODUTO, ao ser clicado, limpa os campos do produto exibido.

Os botões ANTERIOR e PRÓXIMO navegam entre os produtos incluídos para esta venda, podendo ser cadastrados até 6 produtos numa mesma venda.

O botão CREDIARISTA aparecerá quando a venda for a prazo e, ao ser clicado, aparecerá uma listagem de crediarista para que seja escolhido um para o financiamento desta venda. Para escolher um crediarista na listagem basta clicar 2 vezes sobre o nome do crediarista escolhido ou se, o crediarista for o próprio cliente, é só marcar o campo O PRÓPRIO.

Caso a venda seja feita a prazo com cheques, o botão INCLUIR PARCELAS aparecerá para que os cheques recebidos referente as parcelas deste crediário sejam cadastrados.
