Visualizando opções extras
 

A tela abaixo é um exemplo de opções extras existentes no sistema.
As opções são: folhetos explicativos, mala direta, cartas de cobrança, aniversariante e proposta de crédito.
   
.

   
Preparando impressão de folhetos explicativos
 

Para se imprimir um folheto explicativo, basta clicar na opção FOLHETO EXPLICATIVO, abrirá uma opção de consulta para que você possa escolher o folheto que será impresso e clique no botão IMPRIMIR.
   
 
   
Preparando impressão de
mala direta
 

Para se imprimir uma mala direta, basta clicar na opção MALA DIRETA, abrirá uma opção de consulta para que você possa escolher o tipo de mala direta que será impresso e clique no botão IMPRIMIR.
   
 
   
  Uma visualização da impressão aparecerá.
Desta forma poderá ver como ficará a impressão antes de confirmá-la.
   
 
   
Preparando impressão de
cartas de cobrança
 

Para se imprimir cartas de cobrança, basta clicar na opção CARTAS DE COBRANÇA, abrirá uma opção de consulta para que você possa escolher o tipo de carta que será impresso e para que cliente essa carta se destina e clique no botão IMPRIMIR.
   
 
   
  Antes de imprimir, a tela abaixo aparecerá para que você possa personalizar a assinatura e/ou as informações de sua empresa. Para personalizar basta preencher os campos e depois clicar no botão ASSINAR, uma vez feita essa operação, o sistema guardará sempre a última personalização feita, para que não seja preciso personalizar toda vez que precisar imprimir uma carta de cobrança. A assinatura poderá ser facilmente alterada, para isso, basta modificar os dados que serão preenchidos automaticamente na próxima vez que se for imprimir uma nova carta. Caso não queira personalizar, basta clicar no botão BRANCO e nenhuma personalização será feita na carta. O botão BRANCO NÃO apagará nenhuma personalização já gravada anteriormente.
   
 
   
Preparando impressão do relatório de aniversariantes
 

Para se imprimir um relatório de aniversariantes, basta clicar na opção ANIVERSARIANTES, abrirá uma opção de consulta para que você possa escolher a data de aniversário para a impressão e clique no botão IMPRIMIR.
   
 
   
 

Uma visualização da impressão aparecerá.
Desta forma poderá ver como ficará a impressão antes de confirmá-la.
   
 
   
Preparando impressão de uma
proposta de crédito
 

Para se imprimir uma proposta de crédito, basta clicar na opção PROPOSTA DE CRÉDITO, abrirá uma opção de consulta para que você possa pesquisar sua proposta por Cliente, Crediarista e/ou Data do crediário e clique no botão IMPRIMIR.
   
 
   
  Antes de imprimir, a tela abaixo aparecerá para que você possa personalizar a assinatura e/ou as informações de sua empresa. Para personalizar basta preencher os campos e depois clicar no botão PERSONALIZAR, uma vez feita essa operação, o sistema guardará sempre a última personalização feita, para que não seja preciso personalizar toda vez que precisar imprimir uma proposta de crédito. A personalização poderá ser facilmente alterada, para isso, basta modificar os dados que serão preenchidos automaticamente na próxima vez que se for imprimir uma nova proposta de crédito. Caso não queira personalizar, basta clicar no botão EM BRANCO e nenhuma personalização será feita na proposta. O botão EM BRANCO NÃO apagará nenhuma personalização já gravada anteriormente.
   
 
   
  Uma visualização da impressão aparecerá.
Desta forma poderá ver como ficará a impressão antes de confirmá-la.
